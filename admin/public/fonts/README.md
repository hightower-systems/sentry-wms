# Self-hosted admin fonts

Both fonts ship as a single variable woff2 each; the admin panel renders
400/500/600/700 of Instrument Sans and 400 of JetBrains Mono. Source:

- **Instrument Sans** — © 2022 The Instrument Sans Project Authors.
  https://github.com/Instrument/instrument-sans — SIL Open Font License 1.1
  (see `OFL-InstrumentSans.txt`).
- **JetBrains Mono** — © 2020 The JetBrains Mono Project Authors.
  https://github.com/JetBrains/JetBrainsMono — SIL Open Font License 1.1
  (see `OFL-JetBrainsMono.txt`).

Both licenses permit embedding and redistribution unmodified; the OFL
requires the license text to travel with the font file, which is why
the two `OFL-*.txt` files live alongside the woff2 in this directory.
